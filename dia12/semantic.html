Web semántica

Elementos con significado

Ejemplos de etiquetas semánticas: table, form, article Ejemplos de etiquetas no semánticas: div, span...

Header

Es el encabezado de una página, sección o artículo. Incluye elementos como el logo, texto introductorio, menús de navegación. Puede haber múltiples headers en una página, pero no puede haber un header dentro de un footer o de otro header.

Footer

Es el pie de página de un sitio o sección. Puede haber múltiples por página. Puede incluir información como:
  • Información de contacto
  • Enlaces a las RRSS
  • Mapa del sitio
  • Créditos
  • Formulario de contacto

Nav

Define un conjunto de enlaces de navegación. No todos los links deben estar dentro de un nav.

Main

Crea un contenedor para el contenido principal de nuestra página. Este debe ser único y no debe incluir contenido repetido de otras secciones, como por ejemplo del sidebar, logos, etc. No puede ser hijo de etiquetas como article, aside, footer, header o nav. (Al revés sí)

Aside

Define el contenido relacionado indirectamente con el contenido principal, como por ejemplo un sidebar. Elementos que solemos poner en un aside:
  • Publicidad
  • Biografía del autor
  • Enlaces relacionados
  • Información de perfil
A veces se pone oculto (display:hidden) para hacer una web responsive y que desaparezca, por ejemplo, en móviles, a no ser que pulses en algún botón.

Section

Define una sección en una página donde se agrupan los contenidos relacionados, y normalmente ingluyen un título.
Una web se compone de varias secciones, cada una con su contenido y propósito.
  • Capítulos
  • Introducción
  • Noticias o categorías
  • Información de contacto

Article

Define un contenido independiente. El mismo puede ser compartido o reutilizado en otro sitio sin requerir contexto.
También suele incluir un título. Ejemplos de uso:
  • Un post de un blog
  • Un artículo de una revista
  • Un comentario de un usuario (como las reseñas)
  • Tarjeta de producto (por ej. Wallapop)

Puedo encontrar artículos dentro de secciones y secciones dentro de artículos. Una sección puede tener múltiples artículos y un artículo puede tener múltiples secciones.

Figure

Define elementos autocontenidos, como puede ser una imagen, un diagrama, un gráfico, una tabla o una foto.
Dentro de ella, se suele utilizar la etiqueta "figcaption", para describir el contenido (pies de fotos, de tablas...). Esto es además del ya incluido "alt" en caso de imágenes.

Mark

Define un texto resaltado, normalmente se muestra con fondo amarillo
